Point essentiel pour un site web
Son hébergement

Aujourd’hui un point essentiel pour un site internet est son hébergement, de manière évidente s’il n’est pas hébergé, il n’est pas disponible en ligne. Ainsi de nombreux service proposer l'hébergement de site internet avec une gestion de celui-ci et une panopli d’outils associés

type d’hébergement possible :

  • 01

    Hébergement mutualisée

    Principe :
    Consiste à se partager un serveur à plusieurs

    Avantage :
    - Economique
    - Simple d'utilisation

    Inconvénient :
    - Impact sur les performances
    - Pas adapté au forts traffics
    - Configuration bridée
    - Inconvénient pour les utilisateurs avancés

  • 02

    Hébergement dédié

    Principe :
    Location auprès d’un hébergeur web d’un serveur très configurable.

    Avantage :
    - Très personnalisable
    - Améliore les performances
    - Très personnalisable
    - Meilleur sécurité

    Inconvénient :
    - Couteux
    - Charge du client pour les interventions
    - Nécessite connaissance plus avancée en réseaux


  • 03

    Hébergement VPS

    Principe :
    Consiste à la création de serveur virtuelle sur un serveur réel

    Avantage :
    - Bon rapport prix performance
    - Simple d'utilisation - Adaptation possible en fonction des besoins
    - Serveur alimenter comme souhaité

    Inconvénient :
    - Impact sur les performances
    - Pas adapté au forts traffics
    - Nécessite connaissance en réseaux

  • 04

    Hébergement Cloud

    Principe :
    Consiste à se partager un serveur à plusieurs

    Avantage :
    - Grande flexibilité
    - Paiement en fonction des besoins réels
    - Support des variations très bien gérer

    Inconvénient :
    - Impact sur les performances
    - Données stockée hors de l’entreprise
